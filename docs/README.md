# Documentacion Opos Enfermeria App

Este directorio divide la documentacion por temas para facilitar mantenimiento.

## Indice

- [Arquitectura](architecture.md) — Stack, perfiles, modulos clave y flujo de la app.
- [Datos y Persistencia](data-and-storage.md) — Estructura de datos, claves AsyncStorage y cache remota.
- [Sistema de Correcciones](corrections-system.md) — Correcciones locales, envio a GitHub y consenso automatico.
- [Build, Deploy y Operacion](build-and-operations.md) — Comandos de build, generacion de preguntas y troubleshooting.

## Lectura recomendada

1. Leer [Arquitectura](architecture.md) para entender la estructura general.
2. Revisar [Datos y Persistencia](data-and-storage.md) para entender como se almacenan preguntas y progreso.
3. Revisar [Sistema de Correcciones](corrections-system.md) para cambios en el flujo de sugerencias.
4. Consultar [Build, Deploy y Operacion](build-and-operations.md) antes de generar APK o publicar.
